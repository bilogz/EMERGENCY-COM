AlertaraQC is a mobile-based Emergency Communication System designed to help local government units efficiently disseminate critical alerts, warnings, and public announcements. The system focuses on fast, reliable, and structured communication during emergencies.

Overview

The system enables authorities to issue categorized emergency alerts while allowing citizens to receive timely notifications, manage alert preferences, and participate in two-way communication. It supports integration with official warning sources and maintains detailed logs for accountability and review.

Core Modules

Mass Notification System
Delivers alerts through multiple channels such as in-app notifications, SMS, and email to ensure maximum reach.

Alert Categorization
Organizes alerts into predefined categories such as Weather, Earthquake, Fire, and Security Threats to standardize response procedures.

Two-Way Communication Interface
Allows users to acknowledge alerts, send messages, and provide real-time feedback to authorities.

Automated Warning Integration
Supports automated feeds from official agencies such as PAGASA and PHIVOLCS to ensure accurate and timely alerts.

Multilingual Alert Support
Provides multilingual message delivery based on user language preferences.

Citizen Subscription and Preferences
Enables users to subscribe to specific alert types and configure notification methods.

Log and Audit Trail
Maintains detailed records of alerts, notifications, and system activities for transparency and monitoring.
